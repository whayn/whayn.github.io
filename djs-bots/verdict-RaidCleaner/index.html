Lexique

  • [-] Point négatif.
  • [+] Point positif.
  • [?] Suggestion.
  • [~] Point ni positif ni négatif.
  • <> Argument obligatoire (ne pas mettre les <> dans la commande).
  • [] Argument facultatif (ne pas mettre les [] dans la commande).
  • a|b Alias dans le nom d’une commande ou d’un type d’argument.

Analyse

[clean]

  • [+] Efficace et pratique
  • [-] Ne trouve pas de channel si on utilise une mention
  • [?] Cela pourrais être pratique que sa clean en utilisant les mention dans le cas où on a des caractère spécial
  • [?] Ajouter la possibilité d’épargner un channel avec son ID si nous avons parmi tout ces channel un que l’on ne veut pas supprimer

[server-locked]

  • [~] Fait crash le bot

[join-verif]

  • [-] N’enregistre pas les configurations

[anti-spam]

  • [-] Il tente de ban si la personne détient des pouvoir supérieure au sien
  • [?] Si la personne détient des permission supérieure alors un return serait plus efficace

[config/lang]

  • [?] le message disant que la langue a été définie devrait être traduit dans la nouvelle langue et non dans l’ancienne

[config/(warn/anti-spam)]

  • [-] utilisation incompréhensible
  • [-] Quand on spam ça nous renvoie toujours le message pour dire qu’il vient d’être activé (ex: /config warn mute)
  • [-] le off ne revoie aucun message
  • [-] la commande /config list ne fonctionne pas

[warn/tempmute/kick]

  • [-] non fonctionnel

[user-info]

  • [?] Renvoie les infos de l’utilisateur ayant exécuté la commande si aucun user est mentionné

Conclusion

➕ Points positifs :
Présentation esthétique et efficace et beaucoup de fonctionnalités utiles.
Plein de fonctionnalités super pratiques pour protéger son serveur.

➖ Points négatifs :
Beaucoup de commandes sont incompréhensibles et inutilisables/non fonctionnelles.

❓ Suggestions :
Une amélioration du sélecteur d’utilisateur dans les arguments.

Bot très pratique, de nombreuses fonctionnalités très utiles pour protéger son serveur, mais a besoin de nombreuses améliorations et corrections aux nombreux bugs.

À bientôt, n’hésitez pas à revenir et à faire un peu de pub.
