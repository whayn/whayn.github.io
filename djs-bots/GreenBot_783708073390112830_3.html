Lexique

  • [-] Point négatif.
  • [+] Point positif.
  • [?] Suggestion.
  • [~] Point ni positif ni négatif.
  • <> Argument obligatoire (ne pas mettre les <> dans la commande).
  • [] Argument facultatif (ne pas mettre les [] dans la commande).
  • a|b Alias dans le nom d’une commande ou d’un type d’argument.

Conclusion

Très bon bot de musique, contient toutes les fonctions demandé pour bien fonctionner

➖ Points négatifs :
Quelques commandes fonctionnent bizarrement ou juste ne fonctionnent pas, le dashboard est très difficile et désagréable a utiliser pour certaine fonction

❓ Suggestions :
Se concentrer sur une bonne refonte du dashboard, pour le rendre plus esthétique, ergonomique et fonctionnel
